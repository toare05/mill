# 계산기밀(軍) 서비스 PRD (Product Requirements Document)

## 1. 개요

계산기밀(軍)은 대한민국 군 관련 다양한 계산기를 제공하는 웹 애플리케이션입니다. 현재는 공군 지원자들을 위한 1차 점수 계산기와 장병내일준비적금 수령액 계산기를 주요 서비스로 제공합니다. 사용자들이 필요한 정보를 입력하면 실시간으로 결과가 계산되어 표시됩니다.

## 2. 기술 스택

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui 컴포넌트 라이브러리
- React Hooks (상태 관리)

## 3. 공통 UI/UX 요구사항

- Apple 스타일의 디자인 (깔끔하고 미니멀한 인터페이스)
- 직관적인 사용자 인터페이스
- 모바일 및 데스크톱 환경에서 모두 사용 가능한 반응형 디자인
- 사용자 입력 변경 시 실시간 결과 업데이트
- 페이지 새로고침 시 입력 데이터 초기화 (데이터 저장 없음)

## 5. 기능별 요구사항

### 5.1 공군 지원 점수 계산기

#### 주요 기능
- 단일 페이지 애플리케이션으로 모든 기능 제공
- 사용자 입력에 따른 실시간 점수 계산
- 군인 유형별(일반기술병/전문기술병) 점수 체계 적용
- 자격/면허, 전공, 출결, 가산점을 포함한 1차 점수 계산
- 선발 커트라인 비교 정보 제공

#### 점수 계산 로직
**일반기술병(일반직종)**
- 자격/면허: 70점
- 출결: 20점
- 가산점: 15점
- 총점: 105점 (면접 제외)

**전문기술병(화생방, 의무, 기계, 통신전자전기, 차량정비, 차량운전, 전자계산)**
- 자격/면허: 50점
- 전공: 40점
- 출결: 10점
- 가산점: 15점
- 총점: 115점 (면접 제외)

#### 입력 항목
- 군인 유형 (일반기술병/전문기술병)
- 특기 선택 (전문기술병의 경우)
- 자격/면허 정보
- 전공 정보
- 출결 상황
- 가산점 항목
- 지원 월 선택

### 5.2 장병내일준비적금 계산기

#### 주요 기능
- 단일 페이지 애플리케이션으로 모든 기능 제공
- 사용자 입력에 따른 실시간 수령액 계산
- 총 납입액, 이자, 정부 지원금 등 세부 내역 표시
- 복무 기간별 계산 지원
- 은행별 금리 비교 기능

#### 계산 요소
**기본 입력값**
- 월 납입액 (1만원~55만원)
- 복무 기간 (1~24개월)
- 은행 선택
- 정부 매칭 지원금 적용 여부
- 2025년 이후 납입 여부

**계산 기준**
- 은행별 기본 금리 및 우대 금리
- 정부 매칭 지원금: 월 납입액의 1/3 (최대 15만원)

#### 수령액 계산 로직
- 총 납입액: 월 납입액 × 복무 개월 수
- 이자 계산 (단리 적용)
- 정부 매칭 지원금 계산
- 총 수령액: 총 납입액 + 이자 + 정부 매칭 지원금
